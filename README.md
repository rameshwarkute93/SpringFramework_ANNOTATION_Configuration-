# SpringFramework_ANNOTATION_Configuration :


1) Create The Java Project
2) Configure the Required Jar File(Download From Maven Repositery)
3) Create The Packages
   - for main class
   - for Bean/entity
   - for Xml file
   - for Java file
# Work Flow :
- Applicattion Context read Java repository file.
- In Java repository file , Scan tha Bean Compenent using @ComponentScan("cs.model").
- Value are set to bean in bean class using @Value()
- Using application context bean are get and disply.
